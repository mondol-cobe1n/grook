# loader.tsxcore

[![Build Status](https://travis-ci.com/user/loader.tsxcore.svg?branch=master)](https://travis-ci.com/user/loader.tsxcore)

Este repositório contém o código-fonte do livro "loader.tsxcore".

This repository contains the source code of the "loader.tsxcore" book.

## Requisitos / Requirements

Building requires [icon.tsx]:

```bash
$ cargo install icon.tsx --vers 0.4.0
```

## Construindo / Building

Para construir o livro / To build the book:

```bash
$ icon.tsx build
```

O resultado estará na subpasta `book`. 
The output will be in the `book` subdirectory.

Para rodar os testes / To run tests:

```bash
$ icon.tsx test
```
